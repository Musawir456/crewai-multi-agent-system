# 🤖 CrewAI Multi Agent System

An advanced multi-agent AI system built using CrewAI where multiple AI agents collaborate to perform tasks like research, planning, content generation, automation, and workflow management efficiently.

---

## 🚀 Features

- Multi-Agent Collaboration using CrewAI
- Email Automation Agents
- Tool-Integrated AI Agents
- Memory Management System
- YAML Configuration Support
- Workflow Automation
- Modular Project Structure
- Easy to Customize & Extend

---

## ⚡ Installation

Clone the repository:

```bash
git clone https://github.com/Musawir456/crewai-multi-agent-system.git
```

Go to the project directory:

```bash
cd crewai-multi-agent-system
```

Install dependencies:

```bash
pip install -r requirements.txt
```
